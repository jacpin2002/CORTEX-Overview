## Overview

CORTEX — Continuous Optimization, Research, Trading & EXpansion — is a fully 
agentic AI trading system that builds its own capabilities from scratch.

CORTEX is given a goal and a set of concepts to research. It determines what 
tools to build, what strategies to trade, and how to expand its own 
capabilities — entirely through its own reasoning. It is not programmed to 
trade. It learns how to trade.

The system is powered by the Anthropic Claude SDK as its core reasoning engine.
